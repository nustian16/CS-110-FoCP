About Programe:
This is a file handing program. which read a image (ppm exe file), and create a new file with effects which you choices.
this program contains 8 effects.
1.Convert to greyscale 		2.flip horizentally 
3.negative of red 		4.negative of green
5.negative of blue 		6.flatten red 
7.flatten green 		8.flatten blue

this program works on the basic of pixels and color value of pixels.
on applying effect it changes values of colors. in the result effect will be shown on created file.
